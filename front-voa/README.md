# VOA Frontend

## How to setup?
```
1. Build gradle script after you clone this project
2. Install node.js if not insatalled then reboot computer
3. Run command "npm install" in terminal 
```

## How to run?
```
1. Start spring boot
2. cd front-voa
3. Run command "npm run build" or 
   Run command "npm run dev" in terminal for watch
4. After finished the build, re-run spring boot application
5. Refresh web browser - sometimes you need hard refresh
```
